# Geeko Arcade

This is a Vite + Vue project displaying a 3D cube using Three.js.

## Project Setup

The project was scaffolded using `pnpm create vite@latest . -- --template vue`.

## Dependencies

- `vue`
- `three` (Used instead of `@webgamekit/threejs` and `@webgamekit/animation` as they were not found)

## Running the Project

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Run the development server:
   ```bash
   pnpm dev
   ```

3. Build for production:
   ```bash
   pnpm build
   ```
