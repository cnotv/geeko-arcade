# Geeko Arcade

This is a Vite + Vue project displaying a 3D cube using Three.js.

## Project Setup

The project was scaffolded using `npm create vite@latest . -- --template vue`.

## Dependencies

- `vue`
- `three` (Used instead of `@webgametoolkit/threejs` and `@webgametoolkit/animation` as they were not found)

## Running the Project

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
